# Análisis comparativo de tiendas

## Descripción
Este proyecto realiza un análisis comparativo de cuatro tiendas con el objetivo de determinar cuál es la mejor opción para vender productos. El análisis se basa en indicadores como ingresos totales, ventas, calificación promedio de los clientes, costo de envío promedio y comportamiento de productos y categorías más vendidas.

## Objetivo
Recomendar la tienda más conveniente para vender, a partir del análisis de datos y visualizaciones generadas en Python.

## Tecnologías utilizadas
- Python
- Pandas
- Google Colab

## Estructura del proyecto
- Carga y exploración de datos  
- Cálculo de métricas (ventas, ingresos, valoraciones y costos de envío)  
- Generación de gráficos comparativos  
- Cuadro comparativo y ranking general  
- Conclusión 

## Cómo ejecutar el proyecto
1. Abrir el archivo en Google Colab.
2. Ejecutar las celdas en orden.
3. Revisar los gráficos, tablas comparativas y la conclusión final.

## Resultados
A partir del análisis realizado, se identifica la **Tienda 4** como la mejor opción en general, debido a su mayor rentabilidad, volumen de ventas y menor costo de envío promedio.

## Observaciones
Los resultados se basan en los datos proporcionados y pueden variar si se actualiza la información de las tiendas.
